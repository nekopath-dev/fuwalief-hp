================================================================================
                    Irodori プロジェクト概要
================================================================================

最終更新: 2026年2月3日

================================================================================
                    アプリ概要
================================================================================

名前: Irodori（旧名: Dynamic Wallpaper）
種類: Android AI壁紙生成アプリ
収益モデル: リワード広告（広告を見てクレジット獲得 → クレジットで壁紙生成）

================================================================================
                    技術スタック
================================================================================

【Android】
- 言語: Kotlin
- UI: Jetpack Compose + Material3（ダークテーマ）
- アーキテクチャ: MVVM + Clean Architecture
- DI: Hilt
- 非同期: Kotlin Coroutines + Flow
- バックグラウンド: WorkManager
- 広告: Google AdMob (リワード広告)
- ストレージ: DataStore (クレジット管理)

【バックエンド】
- Firebase Cloud Functions (Node.js 20)
- リージョン: asia-northeast1
- 画像生成: Stability AI SD 3.5 (移行済み ✅)
- 翻訳: Gemini 2.0 Flash（日本語→英語プロンプト翻訳）
- ストレージ: Firebase Storage

【開発環境】
- OS: Windows
- IDE: Android Studio（ビルドはAndroid Studioから実行）
- CLI: Firebase CLI

================================================================================
                    主要機能
================================================================================

1. プロンプトから壁紙生成（高速/高品質モード選択）
2. ランダム壁紙生成（カジノ風UI）
3. テンプレートから壁紙生成
4. 壁紙の自動更新（WorkManager）
5. バックグラウンド壁紙生成（WorkManager、アプリを閉じても継続）
6. 壁紙の設定（ホーム/ロック画面）
7. 広告視聴によるクレジット獲得
