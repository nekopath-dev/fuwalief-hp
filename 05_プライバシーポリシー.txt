Irodori プライバシーポリシー
============================

最終更新日: 2026年2月3日
施行日: 2026年2月3日

fuwalief（以下「当方」といいます）は、Irodori（以下「本アプリ」といいます）における
ユーザーの個人情報の取扱いについて、以下のとおりプライバシーポリシー
（以下「本ポリシー」といいます）を定めます。

第1条（アプリ提供者情報）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

アプリ名: Irodori
提供者: fuwalief（個人開発者）
お問い合わせ先: fuwalief@gmail.com

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第2条（収集する情報）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

本アプリでは、以下の情報を収集します。

1. ユーザーが提供する情報
   (1) Googleアカウント情報（氏名、メールアドレス、プロフィール画像URL）
       — Googleアカウントによるログイン時に取得します。

2. 自動的に収集される情報
   (1) デバイス識別子（Android広告ID、アプリセットID）
   (2) IPアドレス
   (3) デバイス情報（OS バージョン、機種名）
   (4) アプリの利用状況（起動回数、機能の使用状況）
   (5) 診断情報（アプリの起動時間、パフォーマンスデータ）
   (6) 位置情報（天気連動機能で使用する場合のみ、おおまかな位置情報）

3. 生成に関する情報
   (1) 壁紙生成時に入力されたプロンプト（テキスト）
   (2) 生成された壁紙の画像データ
   (3) クレジットの利用履歴

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第3条（情報の収集方法）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. ユーザーによる入力
   Googleアカウントでのログイン時、およびプロンプト入力時に、ユーザーが
   直接情報を提供します。

2. SDKによる自動収集
   本アプリに組み込まれた以下のSDK・サービスにより、情報が自動的に
   収集されます。
   (1) Firebase Authentication — ユーザー認証・アカウント管理
   (2) Firebase Cloud Functions — 壁紙生成処理
   (3) Firebase Cloud Storage — 生成画像の保存
   (4) Google AdMob SDK — 広告配信
   (5) Google Analytics for Firebase — アプリ利用状況の分析

3. 端末の機能
   位置情報は、ユーザーがOSレベルで位置情報の利用を許可した場合にのみ
   取得されます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第4条（情報の利用目的）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

収集した情報は、以下の目的で利用します。

1. 本アプリのサービス提供
   — ユーザー認証、壁紙の生成・保存・設定、クレジット管理

2. 広告の配信
   — Google AdMobを通じたリワード広告の表示
   — パーソナライズド広告（ユーザーの関心に基づく広告）の配信
     ※ ユーザーは端末の設定から広告のパーソナライズをオプトアウトできます
     （設定 → Google → 広告 → 「広告のパーソナライズをオプトアウト」）

3. 天気連動機能
   — 位置情報に基づく天気情報の取得と、天気に応じた壁紙生成

4. サービスの改善
   — アプリの利用状況の分析、不具合の検出と修正

5. 不正利用の防止
   — 不正なアクセスや利用の検出・防止

6. お問い合わせへの対応
   — ユーザーからのお問い合わせに対する回答・対応

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第5条（第三者への情報提供）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

当方は、以下の場合を除き、ユーザーの同意なく個人情報を第三者に提供しません。

1. 本アプリの機能提供に必要な業務委託先

   (1) Google LLC（米国）
       — Firebase（認証・データ保存・分析）、AdMob（広告配信）
       — Googleのプライバシーポリシー:
         https://policies.google.com/privacy?hl=ja

   (2) Stability AI Ltd.（英国）
       — AI画像生成エンジンの提供
       — ユーザーが入力したプロンプト（テキスト）を画像生成のために送信します
       — 生成されたプロンプトおよび画像は、Stability AIのサーバーで処理されます
       — Stability AIのプライバシーポリシー:
         https://stability.ai/privacy-policy

2. 法令に基づく場合
   法令に基づき開示が求められた場合。

3. 人の生命・身体・財産の保護に必要な場合
   ユーザー本人の同意を得ることが困難であるとき。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第6条（外国にある第三者への提供）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

本アプリでは、以下の外国にある第三者にデータを提供します。

1. Google LLC（アメリカ合衆国）
   — 個人情報保護制度: セクトラル方式（州法による保護、CCPA等）
   — Googleは標準契約条項（SCC）等に基づくデータ保護措置を講じています

2. Stability AI Ltd.（イギリス）
   — 個人情報保護制度: UK GDPR（EU GDPRと同等の保護水準）

これらの提供は、日本の個人情報保護法第28条に基づき、適切な措置を講じた
上で行われます。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第7条（広告について）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 本アプリではGoogle AdMobを利用してリワード広告を配信しています。

2. AdMobは、広告配信のためにデバイス識別子（Android広告ID）、IPアドレス、
   ユーザーの操作情報（タップ、動画視聴等）を自動的に収集します。

3. 収集されたデータは、広告の配信・効果測定・不正防止のために使用されます。

4. すべてのデータはTLS（Transport Layer Security）により暗号化された
   状態で転送されます。

5. パーソナライズド広告のオプトアウト
   ユーザーは以下の方法で広告のパーソナライズを無効化できます。
   — Android端末: 設定 → Google → 広告 → 「広告のパーソナライズをオプトアウト」
   — Google広告設定: https://adssettings.google.com/

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第8条（データの保存と管理）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Googleアカウント情報およびクレジット情報は、ユーザーの端末内
   （DataStore）に保存されます。

2. 生成された壁紙画像は、Firebase Cloud Storageに一時保存され、
   端末へのダウンロード後に削除される場合があります。

3. ユーザーが入力したプロンプトは、画像生成のためにFirebase Cloud Functions
   およびStability AI APIに送信されますが、当方のサーバーに永続的に
   保存されることはありません。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第9条（ユーザーの権利）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ユーザーは、個人情報保護法に基づき、以下の権利を有します。

1. 開示請求
   当方が保有するユーザーの個人情報について、開示を請求することができます。

2. 訂正・追加・削除請求
   個人情報が事実でない場合、訂正・追加・削除を請求することができます。

3. 利用停止・消去請求
   個人情報が利用目的の範囲を超えて利用されている場合、または不正に取得
   された場合、利用停止・消去を請求することができます。

4. アカウント削除
   本アプリ内の「設定」→「アカウント」→「アカウントを削除」から、
   いつでもアカウントを削除できます。アカウントを削除すると、Firebase
   Authentication上の認証情報が削除されます。

5. データ収集の停止
   本アプリをアンインストールすることで、本アプリによるデータ収集を
   停止することができます。

上記の請求を行う場合は、第1条に記載のお問い合わせ先までご連絡ください。
本人確認を行った上で、合理的な期間内に対応いたします。

第10条（児童のプライバシー）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

本アプリは、13歳未満の児童を対象としていません。当方は、13歳未満の
児童から意図的に個人情報を収集することはありません。13歳未満の児童が
個人情報を提供したことが判明した場合、当方は速やかにその情報を削除します。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第11条（セキュリティ）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

当方は、収集した個人情報の漏洩、紛失、毀損を防止するために、以下の措置を
講じています。

1. 通信の暗号化（HTTPS/TLS）
2. Firebase Authenticationによる認証管理
3. Credential Manager APIによるセキュアなログイン処理
4. アクセス権限の最小化

ただし、インターネット上の通信においてセキュリティを完全に保証することは
できません。
